# Assemblor

## Simple markdown generator from assembla tickets based on milestone

Usage:
### Setup Environment
```bash
export API_KEY=<your key>
```
```bash
export API_SECRET=<your_secret>
```

### Run
```bash
npm install
```
```bash
npm start <milestone_name> <changelog_file>
```

Example: npm start 0.10.0 ./CHANGELOG.md
